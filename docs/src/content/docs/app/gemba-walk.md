---
title: Gemba Walk
description: Structured 5-step shopfloor observation process with findings and issue creation.
---

The Gemba Walk page guides area leaders through a structured 5-step observation process on the shopfloor. It follows the SQDCP methodology (Safety, Quality, Delivery, Cost, People).

## Starting a Walk

When no walk is active, the page shows a form to start a new one.

```
┌─────────────────────────────────────────────────────────┐
│  Management Tools › Gemba Walk                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌─ Start Gemba Walk ───────────────────────────────┐   │
│  │                                                  │   │
│  │  Target Areas                                    │   │
│  │  ┌──────────────────────────────────────────┐    │   │
│  │  │ e.g., Production Area A, Packaging       │    │   │
│  │  └──────────────────────────────────────────┘    │   │
```

## 5-Step Walk Process

Once a walk is started, a progress indicator shows the current step.

```
  ┌────────────┐   ┌────────────┐   ┌───────────────┐   ┌────────┐   ┌────────┐
  │ 1.Initiate │──▶│ 2.Observe  │──▶│ 3.Document    │──▶│4.Issues│──▶│5.Report│
  │  (active)  │   │            │   │               │   │        │   │        │
  └────────────┘   └────────────┘   └───────────────┘   └────────┘   └────────┘
```

### Step 1: Initiation

Confirm or adjust the walk parameters.

```
┌─ Step 1: Initiation ───────────────────────────────────┐
│                                                        │
│  Target Areas                                          │
│  ┌────────────────────────────────────────────────┐    │
│  │ Production Area A, Packaging                   │    │
│  └────────────────────────────────────────────────┘    │
│                                                        │
│  Focus                                                 │
│  ┌────────────────────────────────────────────────┐    │
│  │ Safety compliance, 5S audit                    │    │
│  └────────────────────────────────────────────────┘    │
│                                                        │
│  Participants                                          │
│  ┌────────────────────────────────────────────────┐    │
│  │ J. Doe (Area Leader), K. Smith (Team Lead)     │    │
│  └────────────────────────────────────────────────┘    │
│                                                        │
│                                          [Next >]      │
└────────────────────────────────────────────────────────┘
```

### Step 2: Observation Checklist

An 8-item checklist of standard observations to complete during the walk.

```
┌─ Step 2: Observation ──────────────────────────────────┐
│                                                        │
│  Complete the following observation checklist:          │
│                                                        │
│  [✓] Equipment condition and maintenance status        │
│  [✓] Safety signage and PPE compliance                 │
│  [✓] 5S workplace organization                         │
│  [ ] Production flow and bottlenecks                   │
│  [ ] Quality control checkpoints                       │
│  [ ] Material handling and storage                     │
│  [ ] Team communication boards updated                 │
│  [ ] Standard work procedures followed                 │
│                                                        │
│                              [< Previous]  [Next >]    │
└────────────────────────────────────────────────────────┘
```

### Step 3: Documentation

Free-form text entry for observations and team feedback.

```
┌─ Step 3: Documentation ───────────────────────────────┐
│                                                        │
│  Observations / Findings                               │
│  ┌────────────────────────────────────────────────┐    │
│  │ Area A: Good 5S compliance. One emergency exit │    │
│  │ sign partially blocked by pallet. PPE usage    │    │
│  │ 100% observed. Production board updated.       │    │
│  │                                                │    │
│  │ Packaging: Material staging area needs cleanup.│    │
│  │ Two operators not wearing safety glasses.      │    │
│  └────────────────────────────────────────────────┘    │
│                                                        │
│  Team Feedback                                         │
│  ┌────────────────────────────────────────────────┐    │
│  │ Team requests better lighting at station 5.    │    │
│  │ Positive feedback on new tool organization.    │    │
│  └────────────────────────────────────────────────┘    │
│                                                        │
│                              [< Previous]  [Next >]    │
└────────────────────────────────────────────────────────┘
```
