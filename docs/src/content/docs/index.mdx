## Architecture

The Gemba Management System is designed as a **cloud-native application** deployable via **Docker** and **Kubernetes**, with **ArgoCD** for GitOps-based continuous delivery.

- **Frontend:** React + TypeScript + Vite
- **Backend:** Node.js (Express/Fastify) or Go
- **Database:** PostgreSQL 16
- **Cache:** Redis 7
- **Deployment:** Helm + ArgoCD on Kubernetes
